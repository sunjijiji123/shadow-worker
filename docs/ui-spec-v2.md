## 1. 配色系统(全局统一)

### 1.1 基础色板(深色主题)

```
背景层:
  --bg     #18181B   应用主背景(近黑)
  --bg2    #232327   侧边栏/卡片间背景
  --bg3    #2C2C31   卡片背景
  --rule   #3F3F46   边框/分隔线

文字层:
  --ink    #F4F4F5   主文字(亮)
  --muted  #9CA3AF   次要文字(标签/说明/时间戳)

强调色:
  --accent         #10B981  绿(accent-dim #059669)
                   ← v2 用绿,不是蓝!所有选中/激活/按钮主色用绿
  --danger         #EF4444  红(删除/错误/警告徽章)
```

**关键:v2 用绿色作 accent(#10B981),与 coding 类别色撞色,但角色不同。**
- accent 绿 = UI 操作色(选中态/主按钮/状态就绪)
- coding 蓝(#3B82F6) = 类别色(timeline/统计)
- 不要混用。

### 1.2 类别固定色(timeline/统计/卡片统一)

```css
.coding  { background: #3B82F6; }  /* 编程 */
.office  { background: #8B5CF6; }  /* 办公 */
.browser { background: #F59E0B; }  /* 浏览 */
.chat    { background: #10B981; }  /* 沟通 ← 注意:与 accent 同色值 */
.other   { background: #6B7280; }  /* 其他 */
.idle    { background: #9CA3AF; }  /* 离开/静止 */
```

### 1.3 事件类型色(timeline 上方标记 + 事件列表)

```
voice     #10B981 绿   ▲ 语音
prompt    #F59E0B 橙   ◆ 提示词注入
screenshot #3B82F6 蓝  ◉ 截图
```

## 3. 概览页(Overview)

### 3.1 布局

```
┌──────────────────────────────────────────────────────┐
│ 概览                       [暂停采集] [刷新]           │
├──────────────────────────────────────────────────────┤
│ ┌──────────┐ ┌──────────┐ ┌──────────┐               │
│ │   127    │ │    14    │ │  Cursor  │  ← stats-row  │
│ │ 今日活跃  │ │ 活动段数  │ │ 当前应用  │    3 个等宽卡 │
│ │   分钟    │ │          │ │          │               │
│ └──────────┘ └──────────┘ └──────────┘               │
│                                                      │
│ ┌──────────────────────────────────────────────────┐ │
│ │ 运行状态                                           │ │
│ │ ● 采集中   ● ASR 就绪   ○ VLM 关闭   ● MCP 就绪   │ │
│ └──────────────────────────────────────────────────┘ │
│                                                      │
│ ┌──────────────────────────────────────────────────┐ │
│ │ 今日应用排行                                       │ │
│ │ ● Cursor        coding · 82 min                   │ │
│ │ ● Chrome        browser · 31 min                  │ │
│ │ ● WeChat        chat · 14 min                     │ │
│ └──────────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────────┘
```

### 3.2 元素

**stats-row(3 个等宽统计卡):**
- 今日活跃分钟(大数字 32px 700 字重)
- 活动段数
- 当前应用(文字)

**运行状态卡:**
- 4 个状态点 + 文字(采集中/ASR/VLM/MCP)
- 绿点 ● = 运行/就绪,灰点 ○ = 关闭/未启动

**今日应用排行卡:**
- app-row 列表,每行:圆点(类别色)+ 应用名 + "类别 · 时长"
- 按时长降序

### 3.3 顶部操作

- [暂停采集] ghost 按钮 → 调 `CollectionService.Pause`,变恢复
- [刷新] ghost 按钮 → 重新拉 `GetOverview`

### 3.4 数据来源

gRPC `OverviewService.GetOverview` + `WatchOverview`(stream 实时推)

## 4. 时间轴页(Timeline)

### 4.1 顶部

```
时间轴    [今天 ▾]  [日][周][月]  [搜索事件_____]
          ↑ select  ↑ chip-group  ↑ input
```

- select:今天/昨天/近7天/近30天
- chip-group:日/周/月视图切换
- 搜索框:搜事件文本

### 4.2 日视图

```
2026-06-18 周四    工作 3h 20m · 活跃 4 段

09:00 10:00 11:00 12:00 13:00 14:00 15:00 16:00 17:00 18:00
                                                        ↑ ruler

  ▲      ◆         ◉              ▲                      ↑ events 层
(语音)(提示词)(截图)              (语音)                  在 track 上方

[coding ████████][idle ██][coding ███████][chat ██][browser ████][coding ██]
 ↑ 横向单焦点轴,按时间段拼色块,类别固定色,idle 灰间隙

图例:■编程 ■沟通 ■浏览 ■办公 ■其他 ■离开    ▲语音 ◆提示词 ◉截图
```

**规则(锁定):**
- 单焦点轴(不开多行)
- 色块按类别上色(第 1.2 节)
- idle 是灰色间隙段(.idle class)
- 事件标记在 track **上方**,鼠标 hover 显示 title(时间+内容)
- ruler 时间刻度横排

## 5. 设置页(Settings)

### 5.1 顶部 tab-strip(5 个 tab)

```
语音输入 | 图像识别 | 文字润色 | 个人提示词 | 快捷工具
   ●                                       ← active 下划线 --accent
```

### 5.2 语音输入 tab

**4 张卡:**
1. **ASR 模型服务** — 子 tab(本地模型/云端模型)
   - 本地:chip 选 ggml-tiny/base/+添加,表单(模型名/语言/路径)
   - 云端:chip 选 小米MiMo/智谱GLM/+添加,表单(名称/模型/Base URL/API Format/认证/API Key)+[测试连接]
2. **音频设备** — 输入设备下拉 + [测试麦克风] + 音量条
3. **热词** — textarea 每行一个
4. **录音热键** — 修饰键 select + 按键 input + radio(按住说话/切换模式)
5. **注入模式** — radio(气泡预览/自动注入)

### 5.3 图像识别 tab

**4 张卡:**
1. **VLM 模型服务** — chip(OpenAI兼容/Ollama/Anthropic/+添加),表单(部署类型 云端/本地 + Base URL/模型/API Format/认证/API Key)+[测试连接]
2. **行为采集范围** — radio(整个屏幕/仅活动窗口),活动窗口模式下显示**屏蔽窗口**列表(WeChat/网易云等,选窗口遮罩添加)
3. **采集参数** — 采样间隔/空闲超时/精度(low/medium/high)
4. **VLM 屏幕理解** — chip(关闭/定时截图/按需截图)+ 定时间隔

### 5.4 文字润色 tab

**2 张卡:**
1. **自动润色** — toggle 开关 + 说明
2. **LLM 模型服务** — chip(DeepSeek/OpenAI兼容/Ollama/+添加),表单同 VLM
3. **润色提示词** — textarea(系统默认 + 用户自定义)

### 5.5 个人提示词 tab

**2 张卡:**
1. **快捷提示词注入** — toggle 开关 + 前缀键 select(Ctrl/Alt/Win)
2. **个人提示词列表** — 每行:名称 input + 快捷键 input(1位 0-9/A-Z)+ [删除],[+ 添加]

### 5.6 快捷工具 tab

**2 张卡:**
1. **桌面截图** — 修饰键 + 按键 + 保存位置 + [立即截图]
2. **数据管理** — [打开数据目录] [清空数据]

### 5.7 底部 footer-bar(所有 tab 共享)

```
                              [重置默认]  [保存]
```

## 6. 系统页(System)

**4 张卡:**

## 7. 录音气泡 & 识别结果气泡

### 7.1 录音中气泡(频谱)

```
┌────────────────────┐
│ ▮▮▮▮▮▮▮  录音中     │   ← 圆角 24px,--bg3 背景
└────────────────────┘
wave-bar 4px 宽,--accent 绿,实时跳动
```

### 7.2 识别结果气泡(可编辑)

```
┌──────────────────────────────────┐
│ Enter 注入 · Esc 关闭       ✨    │  ← bubble-hint + AI 润色图标
│ 请把这份需求文档整理成结构化的      │
│ 会议纪要。                       │  ← textarea 可编辑
│              [复制]  [关闭]       │
└──────────────────────────────────┘
```

**AI 润色图标(右上角):**
- 自动润色开启时:图标高亮(--accent 绿,不可点击)
- 自动润色关闭时:图标灰(--muted),可点击手动润色
- 手动润色成功后:高亮不可回退
- hover tooltip:"AI 润色"

**操作:**
- Enter = 注入(到焦点输入框)
- Esc = 关闭气泡
- [复制] [关闭] 按钮

## 8. 通用组件规范

### 8.1 卡片(.card)

```
背景:   --bg3 #2C2C31
边框:   1px solid --rule #3F3F46
圆角:   10px
内边距: 20px
间距:   卡片间 16px
```

### 8.2 按钮

```
.btn-primary  背景 --accent 绿,文字白
.btn-ghost    透明背景,1px --rule 边框,文字 --muted
.btn-danger   透明背景,1px --danger 边框,文字 --danger
.btn-sm       padding 5px 10px,字号 12px
默认           padding 8px 16px,字号 13px,圆角 6px
```

### 8.3 输入框

```
.input / .textarea
背景:   --bg
边框:   1px solid --rule
圆角:   6px
.input 高 36px
.textarea min-height 100px,可调
```

### 8.4 下拉(.select-wrap)

```
触发器:--bg 背景,1px --rule 边框,右侧 ▼ 箭头
选项:--bg 背景,1px --rule 边框,圆角 6px
选项 hover:rgba(16,185,129,0.1) 背景 + 文字 --ink
最大高度 180px,可滚
```

### 8.5 Chip(chip-group)

```
padding 6px 14px
1px --rule 边框,圆角 6px
默认:文字 --muted
hover:边框 --accent
active:边框 --accent + 文字 --accent + 背景 rgba(16,185,129,0.1)
.add 样式:dashed 边框
.star(内置 provider):带星标,不可删
```

### 8.6 Toggle 开关

```
宽 42px 高 22px,圆角 11px
关:背景 --rule,圆点在左(left 2px)
开:背景 --accent,圆点在右(left 22px)
圆点 18px 白色
```

### 8.7 Radio

```
radio-dot 14px,2px --muted 边框
选中:边框 --accent,内部 8px --accent 圆点 + 文字变 --ink
未选:文字 --muted
```
